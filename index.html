    ledlight

    Micro library for Led driving via controllers working under Arduino enviroment. Based on built-in function analogWrite(), focused on optical correction of light using gamma maps and non-linear functions.

    Basic Usage

    Add a library to your project:

    
#include "LedLight.h"

    Add an LedLight Object into your project, uisung PWM PIN number as an argument in brackets().

    
LedLight led(14);

    Put in loop section show function of an object using two arguments: boolean state of an output and brightness value.

    
led.show(bool State, uint16_t Brightness);
